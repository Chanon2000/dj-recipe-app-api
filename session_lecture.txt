        # Adding and listing tags

Tag Model
    - name - Name of tag to create
    - user - User who created/owns tag

Tag Endpoint
    - POST - Create tag
    - PUT/PATCH - Update tags
    - DELETE - Remove tag
    - GET - List available tags
